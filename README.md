```

```

or install manually

```
pip install flask

pip install requests
pip install aiohttp
```

## A quickstart


for Windows

```
$ set FLASK_APP=run.py
$ flask run
```

for Linux

```
$ export FLASK_APP=run.py
$ flask run
```

Service is running on http://127.0.0.1:5000/ by default.

For more information: http://flask.pocoo.org/docs/1.0/quickstart/#
